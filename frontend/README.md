# HN Stories — Frontend

The React-based dashboard for the Hacker News Reader.

## 🚀 Features
- **Independent Scrolling Panes**: List and Preview areas scroll separately.
- **Split-Pane Design**: Optimized for desktop viewing with side-by-side layout.
- **Mobile Overlay**: Focused reading mode for smaller devices.
- **Tailwind CSS v4**: Modern, high-performance styling.
- **React Router 7**: Sophisticated navigation with protected routes.

## ⚙️ Development
```bash
npm install
npm run dev
```

## 🏗️ Tech Stack
- React 19
- Vite 8
- Tailwind CSS 4
- Axios
- React Icons
- React Router 7
